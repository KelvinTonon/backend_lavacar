DESCRIÇÃO LAVACAR
O Lavacar é um projeto desenvolvido em Lavarel, uma aplicação web em PHP,  voltado para dono de empresa de lavacar que busca gerenciar seu perfil, a ideia é uma integração com flutter que será o usuário, no projeto o admin da empresa,fará login/cadastro e poderá fazer cadastro da sua empresa, criar serviços, criar agendamentos, visualizar e edita-los.
CLONAR O REPOSITÓRIO:
git clone https://github.com/KelvinTonon/backend_lavacar.git
cd backend_lavacar
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
